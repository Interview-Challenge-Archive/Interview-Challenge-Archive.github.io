OK, so what is this?

This is a place to show what tests for getting a job looks like in different companies and for different positions. You can either see same such examples here or share own.

Every example that can be found here is hosted on GitHub. So everyone who would like can contribute or use the code elsewhere.

Submit your work

If you ever did a test for a job you can submit into our database here by filling the form. Just few notes before you do so:

  • Make sure you haven't signed any NDA to not publish your test.
  • Make sure that all your files doesn't have parts that can cause any legal issues.
  • You need create a git repository on GitHub first.
  • You should place there all your files.
  • Once you fill the form and submit, your repository will be forked to JobTestVault organization.
